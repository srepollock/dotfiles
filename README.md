# dotfiles

The Dotfiles for my systems
* windows
* macos
* linux

I hope you enjoy them.

## Install

To install the repo and use these dotfiles yourself, clone the repo to your machine and run the `install` script.

```
git clone https://github.com/srepollock/dotfiles
cd dotfiles
./install
```
